# 📋 To-Do List

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.

## ✨ Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🎯 Purpose
This project was built to practice front-end web development and create a simple productivity application.
